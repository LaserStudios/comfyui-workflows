## 🇬🇧 English

Ready-to-load ComfyUI workflows accompanying the tutorials on
[laser-studios.de/tutorials](https://laser-studios.de/tutorials).

Each workflow belongs to a written step-by-step guide. The short video brings the reach,
the tutorial page carries the instructions, and this repo stores the workflow files for
download.

### How to load a workflow

1. Open the relevant folder (e.g. `basics/`).
2. Click the `.json` file, then **Download raw file** (or use the download button at the
   end of the matching tutorial page).
3. In ComfyUI: **Workflow → Open** and select the `.json` — or just drag & drop the file
   into the ComfyUI window.

### Structure

| Folder    | Contents                                                  |
|-----------|-----------------------------------------------------------|
| `basics/` | Foundational workflows – from setup to your first image   |

More categories (e.g. `image-to-video/`, `upscaling/`) will be added once enough
tutorials exist.

### Versioning

Workflow changes go through normal commits. The current version always lives on the
`main` branch; older states are available via the commit history.

### License

Free to use for your own projects. Questions or feedback? Reach out via the Laser Studios
channels.

## 🇩🇪 Deutsch

Fertige, sofort ladbare ComfyUI-Workflows als Begleitmaterial zu den Tutorials auf
[laser-studios.de/tutorials](https://laser-studios.de/tutorials).

Jeder Workflow gehört zu einer schriftlichen Schritt-für-Schritt-Anleitung. Das Video
holt die Reichweite, die Tutorial-Seite trägt die Anleitung, dieses Repo speichert die
Workflow-Dateien zum Download.

### So lädst du einen Workflow

1. Öffne den passenden Ordner (z. B. `basics/`).
2. Klicke auf die `.json`-Datei und dann auf **Download raw file**
   (oder nutze den Download-Button am Ende der jeweiligen Tutorial-Seite).
3. In ComfyUI: **Workflow → Open** und die `.json` auswählen –
   alternativ die Datei per Drag & Drop ins ComfyUI-Fenster ziehen.

### Aufbau

| Ordner    | Inhalt                                                       |
|-----------|--------------------------------------------------------------|
| `basics/` | Grundlagen-Workflows – Einstieg bis zum ersten eigenen Bild  |

Weitere Kategorien (z. B. `image-to-video/`, `upscaling/`) kommen dazu, sobald genügend
Tutorials existieren.

### Versionierung

Änderungen an Workflows laufen über normale Commits. Die jeweils aktuelle Version liegt
immer im `main`-Branch. Ältere Stände lassen sich über den Commit-Verlauf nachvollziehen.

### Lizenz

Frei nutzbar für eigene Projekte. Bei Fragen oder Feedback gern über die Kanäle von
Laser Studios melden.
